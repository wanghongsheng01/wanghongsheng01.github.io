📝 About SciTemplateTrans

背景

在学术论文投稿过程中,研究人员经常需要将论文从一个期刊/会议格式转换到另一个格式。这个过程通常需要花费数小时手动调整格式、重新编排参考文献、调整图表布局等。SciTemplateTrans就是为了解决这个痛点而开发的工具。

真实用例

项目基于一个真实的转换需求场景开发:将一篇ECCV 2024会议论文(使用llncs.cls模板,25条BibTeX引用)转换为Intelligence & Robotics期刊格式。该场景涵盖了学术论文转换的所有典型需求:

  • LaTeX模板切换(会议模板 → 期刊模板)
  • BibTeX引用格式转换
  • 引用位置和编号保持
  • 图表和公式环境适配
  • 版本管理和追踪

技术亮点

  • 智能解析引擎: 基于python-docx、PyPDF2和pylatex的多格式解析
  • 双向转换: Word↔LaTeX互转,LaTeX模板切换
  • 实时预览: 双栏对比,同步滚动,差异高亮
  • 异步处理: Celery后台任务,5阶段进度追踪
  • 微服务架构: FastAPI + React + PostgreSQL + Redis
  • 容器化部署: Docker Compose一键启动

快速开始

git clone https://github.com/wanghongsheng01/SciTemplateTrans
cd SciTemplateTrans/versions/v1.0.0/code
docker-compose up -d

# 前端: http://localhost:3000
# 后端API: http://localhost:8000
# API文档: http://localhost:8000/docs
